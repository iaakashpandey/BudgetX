
# BudgetX

BudgetX is a daily budget management Android application built using Kotlin and MVVM architecture. The app helps users track, manage, and update their daily expenses with ease through a clean and intuitive user interface.

It uses Room Database for offline-first local storage and Firebase for cloud-based data synchronization, ensuring data reliability and accessibility across sessions.

This project demonstrates modern Android development practices, clean code structure, and a user-focused design approach.



## Features

Add, edit, and delete daily expenses

Track daily spending easily

Offline-first support with Room Database

Cloud synchronization using Firebase

Clean and minimal UI

MVVM architecture for scalability


## Tech Stack

Tech Stack

Language: Kotlin

Architecture: MVVM

UI: XML (Material Design)

Local Database: Room

Backend / Cloud: Firebase

Build System: Gradle
